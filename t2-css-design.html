Inline vs. Inline block

8/10/15

Inline is an attribute that is used in CSS, cascading style sheets. Inline is used the same way letters are typed out, one after another across a line. Whereas inline block, not only has the same function as inline, but it also behaves like a block too. Inline block is capable of setting such as width, height, top, bottom, margins, and paddings.
